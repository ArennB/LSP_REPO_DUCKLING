# answers.md

My perspective on using AI in programming has changed slightly over the semester. While I was never hesitant to use it for schoolwork, I wasn't always a big fan of using it for personal projects. I had always felt as though it was taking away from some of the hands-on skills I could be developing from coding "from scratch," and that I wouldn't be as sharp as I could be. However, over time this feeling slowly started to fade. I began to realize that if the entire industry is using AI to their advantage and it is being pushed as a standard workflow, then it makes sense for me to use it when building my own projects as well. Moving forward, I plan to use AI as a tool to support my programming rather than replace my thinking. I want to use it to debug issues, understand difficult concepts, and improve efficiency while still making sure I understand the logic behind my code. I now believe that the best programmers know how to leverage AI to increase productivity, not rely on it to do all the work for them.
